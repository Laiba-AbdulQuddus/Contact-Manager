# Contact Manager

A Python-based command-line application to manage your personal contacts.  
Store, view, search, update, and delete contacts easily.

## Features
- Add new contacts (Name, Phone Number, Email)
- View all contacts in a tabular format
- Search contacts by name
- Update contact information
- Delete contacts
- Data is stored persistently in a JSON file

## Technologies Used
- Python 3
- JSON for storage
- tabulate for pretty-printing tables
